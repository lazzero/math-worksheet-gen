# Math Worksheet Generator

A single-file, static HTML app that generates printable arithmetic worksheets (RU/EN UI, light/dark theme). Designed to quickly create multiple different pages for a whole class and print on A4 or save as PDF.

## Features

- Operations: addition, subtraction, multiplication, division
- Expression structure: 2 numbers, 3 numbers, or mixed
- Blank position: answer only, numbers only, first, middle, or random
- Options: allow negative answers, no-carry addition, no-borrow subtraction
- Per-operation ranges (+, −, ×, ÷)
- Worksheet layout: problems per page, columns, multiple pages
- Deterministic generation via seed (useful for repeating a worksheet set)
- Share settings via URL (Copy link button)
- Print A4 / Export PDF (uses the browser print dialog)
